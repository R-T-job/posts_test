どうでもいいねSNSを作っていた
====
Overview

# プロジェクト概要  

どうでもいいことを投稿するSNSです<br>
### URL
[どうでもいいねSNS](https://dodemo-fav.herokuapp.com/)  <br>

## 環境

Rails 5.2.1<br>
Bootstrap 4<br>
MongoDB v4.0.2<br>

### 経緯
Progate Railsコースを終わったので、<br>
復習も兼ねて、似たようなアプリを作ってみました<br>
<br>
<br>
#### 機能
ユーザ登録機能<br>
→登録後にユーザ登録ボタンからユーザ登録できます<br>
<br>
ログイン機能<br>
→ユーザ登録後にログインできます<br>
<br>
投稿機能<br>
→tweetできます<br>
<br>
ユーザ情報編集、および削除機能<br>
→自分のユーザおよびこのアプリの管理者はプロフィール編集ができます。<br>
またユーザ削除ができます<br>
<br>
ユーザ一覧の一覧が見れます
